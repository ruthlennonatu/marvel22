1. Oracle
Advantages
 - Good for backup and recovery, easily able to make a point-in-time recovery
 - High Performance, good speed with large databases
 - Security, Oracle monitors and control data access and usage. Easy to prevent unauthorized access.
Disadvantages
 - Cost of Oracle, prices can increase up to 10 times compared to MS SQL Server
 - Complexity, Oracle is very complex so not advisable if seeking a database that is easy to use


2. MS SQL Server
Advantages
 - Career opportunities
 - Free developer edition (with all enterprise features)
 - Thriving online community
 - Plenty of online documentation
Disadvantages
 - Complex performance tuning features
 - No native support for source control
